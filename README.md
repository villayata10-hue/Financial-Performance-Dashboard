# **Project Title**

## **Financial Performance Dashboard**


This project presents an interactive Financial Performance Dashboard developed in Tableau to analyze revenue, profitability, cost efficiency, discount impact, and regional performance.

The dashboard is structured into five analytical views: Executive Overview, Country Performance, Product & Segment Analysis, Discount & Pricing Analysis, and Cost & Target Monitoring.

It enables data-driven decision-making through KPI tracking, trend analysis, and interactive visualizations.


Tableau Link (Project) :- https://public.tableau.com/views/FinancialPerformanceDashboard2/Dashboard3?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link

**Objectives**

Monitor overall financial health

Analyze country-wise profitability

Evaluate product performance

Measure discount effectiveness

Track cost efficiency and sales targets



**Key Visualizations**

Line Charts (Sales Trends)

Dual Axis Charts (Sales vs Profit)

Bar Charts (Country & Segment Analysis)

Treemaps (Product Revenue Distribution)

Heat Maps (Discount Band Analysis)

Scatter Plots (Gross Sales vs Discounts)

Waterfall Charts (Profit Breakdown)

Bullet Charts (Sales vs Target)

Map Visualizations (Country Revenue)
